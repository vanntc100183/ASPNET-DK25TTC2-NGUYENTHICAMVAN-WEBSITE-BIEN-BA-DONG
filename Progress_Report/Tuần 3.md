# BÁO CÁO TIẾN ĐỘ ĐỒ ÁN - TUẦN 3
**Đề tài:** Xây dựng website Quảng bá du lịch biển Ba Động
**Thời gian thực hiện:** 06/07/2026 - 12/07/2026
**Giai đoạn:** Thiết lập Project và Xây dựng giao diện (Front-end)

## 1. Nội dung công việc
- Khởi tạo Solution ASP.NET MVC 5 trong Visual Studio, cấu hình cấu trúc thư mục phân định rõ ràng các phân hệ (Areas): Area Admin (Quản trị) và Client (Người dùng).
- Cấu hình kết nối cơ sở dữ liệu trong file `web.config` sử dụng Entity Framework (Database First hoặc Code First).
- Cắt và ghép giao diện Template SB Admin 2 vào cấu trúc Layout của Area Admin; thiết lập các Partial View cho Sidebar, Topbar, Footer.
- Thiết kế giao diện trang chủ Client hiển thị cảnh quan biển Ba Động, danh mục tin tức, sự kiện nổi bật bằng HTML/CSS/Bootstrap.
- Xây dựng các trang tĩnh: Giới thiệu lịch sử, vị trí địa lý, hướng dẫn đường đi đến khu du lịch Ba Động.

## 2. Tài liệu liên quan
- Tài liệu hướng dẫn cấu hình Areas và Routing trong ASP.NET MVC.
- Bộ mã nguồn giao diện HTML/CSS tĩnh đã thiết kế trước và bộ icon FontAwesome.
- Thư viện tối ưu hóa tài nguyên phần cứng mã nguồn mở (Bundling và Minification).

## 3. Khó khăn gặp phải
- Gặp lỗi xung đột đường dẫn (Routing) khi cấu hình các Area trùng tên Controller với phân hệ chính bên ngoài.
- Việc đồng bộ hóa CSS giữa giao diện người dùng và giao diện quản trị đôi khi bị ghi đè, làm vỡ layout hiển thị trên một số thiết bị di động.

## 4. Kết quả đạt được
- Khởi tạo và cấu hình thành công cấu trúc dự án chuẩn, kết nối thông suốt với SQL Server thông qua Entity Framework.
- Hoàn thành thiết kế giao diện tĩnh (UI) cho 100% các trang quản trị và giao diện cốt lõi của trang Client.
